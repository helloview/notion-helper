# 题型一：听线索猜牌

## 定版范例

画面：依次弹出四个图标：一只狗、一只狼、一只小龙虾、两座塔。然后出四个选项：A 星星 / B 月亮 / C 太阳 / D 恶魔。
口播：第一题，一只狗，一只狼，一只小龙虾，还有两座塔。这是哪张牌，五秒的时间，看看你行不行。
答案口播：答案是B，月亮。我猜你一定答对了啦。

## 出题公式

1. 挑一张牌，报三到四个牌面真实元素当线索。**最怪、最萌、最反差的元素放最前面**（小龙虾式），它就是这道题的钩子。
2. 口播只报元素，不报牌名；**选项不读，只进画面**，节奏优先。
3. 作答五秒（默认值，用户可改）。
4. 答案口播 = 答案是X，牌名 + 软夸或软安抚一句（定版：我猜你一定答对了啦） + 可选一个能去搜的真细节，制造去查了居然是真的的验证冲动（例：池塘边真的趴着一只小龙虾，不信去搜）。可选细节一期最多用一次，别题题都喊去搜。

## 干扰项规则：同家族才骗得到人

错误选项必须和正确答案气质相近，一眼能排除的干扰项等于白给。
- 天体家族：星星、月亮、太阳
- 灾难家族：高塔、死神、审判
- 双人家族：恋人、圣杯二、节制
- 老人家族：隐者、教皇、皇帝
- 有动物出场：愚人、月亮、力量、死神、太阳

4. 女祭司：柱子上一个字母B一个字母J、挂满石榴的帘子、一黑一白两根柱子、脚边一弯月牙
5. 皇后：心形的盾牌、十二颗星星的头冠、一片麦田
6. 皇帝：公羊头装饰的王座、一手权杖一手圆球、白胡子老人

宝剑十（背插十剑）画面偏重，轻松题慎用。

## 线索变体（每期换玩法，别一直报元素）

1. 数数线索：这张牌上一共有几个人、几个杯子、几颗星。
2. 颜色线索：只报颜色，一匹白马、一面红旗、一片金黄，猜是哪张。
3. 白话场景线索：把牌面说成一句人话，一个人背着小包在悬崖边遛狗，是哪张牌。
4. 局部特写线索：画面只放牌面一角逐步放大，口播报你看到的部位。

## 查重

每期换牌换线索，不与往期重复。首发期已用：月亮。

## 口播TTS铁律（每句都要过这一关）

1. 全程零引号：中英文单双引号、书名号、括号都不行。
2. 禁波浪号、省略号、破折号、冒号。标点只用逗号、句号、问号、顿号。
3. 数字全写汉字（五秒、十二颗）。选项字母A B C D可以用。
4. 问句必须问号结尾，TTS自动读升调。
5. 生僻字换白话：螯虾写成小龙虾。
6. 倒计时位留给剪辑，口播文本里不写。
7. 空场面话零容忍（别瞎蒙、我不拦着这类），每句话要么装一个真细节，要么埋一个钩子。
8. 语气：大白话短句，亲和带一点点撩，啦、呢、嘛少量点缀。受众约八成是二十五到三十五岁的都市女性打工人。
