# 整期总控：塔罗五题挑战视频

## 定位

一期成片约两分钟：开场钩子 + 四道轻松互动题 + 一道评论区开放题 + 结尾CTA。
前四题负责让观众零门槛玩起来，第五题负责留人、引评论、喂下一期的选题。

## 生成流程（五步，顺序不能乱）

1. **查重**。先从对话历史或用户提供的清单里确认往期已用素材：用过的牌、题型内容、假牌名、第四题考过的牌、以及**第五题的情绪母题**。新一期全部换新，一个都不许重复。**第五题的查重重点不是「换了哪个游戏」，而是「换了哪种情绪母题」**——恋爱暧昧、职场抉择、友情裂痕、家庭、自我、金钱、人生岔路，母题不与最近两期重复才算真换。查不到就先问用户一句：上一期用了什么。
   首发期基线（已用，不可再用）：月亮线索题、太阳记忆题、假牌“预言家”、老板画饼选字幕题（圣杯七+星币八，旧玩法已废弃）、第五题恋爱暧昧母题（和平精英、小鹿、暗恋撤退、圣杯二正/月亮逆/星币四正）。若做过光遇/念念那期演示，其情绪母题同为恋爱暧昧，本期第五题必须换别的母题。
2. **逐段生成**。按顺序套用对应技能的规则：tarot-quiz-hook → q1-clue-guess → q2-flash-memory → q3-fake-card → q4-card-meaning → q5-open-question → tarot-quiz-ending。**第五题先做输入判定**：用户这条消息里给了粉丝真实问题或私信截图，就走来信模式（匿名化改写、忠实不编、优先采用）；什么都没给才走虚构模式（强制轮换情绪母题）。**动笔前，本期用到的每一个牌面细节都必须web搜索核实**，这套号的人设是讲得对，一处硬伤全片报废。
3. **全篇过TTS铁律**（见文末），逐句检查引号、符号、数字写法。
4. **组装输出**（格式见下）。
5. **末尾附史实核查清单和热点·可换提示**。

## 难度曲线（固定）

眼力 → 记性 → 冷知识 → 读牌义 → 开放解读。
这是一条「你到底多懂塔罗」的进阶线：看图、记性、认名字都是表层，第四题考你懂不懂一张牌真正的意思，第五题彻底放开让观众自己解。前四题仍要轻松、扫一眼能玩；只有第五题往情感深处走。作答时间默认每题五秒。

## 时长预算（全片约一分五十）

- 开场钩子：约六秒
- 题一到题四：每题十五到二十秒（口播六到十秒 + 作答五秒 + 答案口播四到六秒）
- 题五：约三十五秒，全片最长段，故意的，它是留人和引评论的段
- 结尾CTA已含在题五内

## 跨题呼应规则

- 默认各题相互独立。可选闭环：只有当开场钩子用了承诺词（牛），才在第五题把它收回来（这题答得好，才算真的牛）；换了不带牛的钩子（比如悬念型），就不收这个闭环，别硬凑。开场若埋了某个牌面细节当悬念（比如着火的树），由第一题兑现即可，这属于钩子对第一题的合法承接，不算跨题埋梗。
- 跨题埋梗默认禁用（比如第一题的元素第五题回收）。定版打磨时的血泪教训：把感情故事里的人比成第一题的小龙虾，气氛当场垮掉。除非气氛百分百匹配，否则不埋。

## 交付格式（定版，四件套）

1. **纯口播复制块**：全部口播按顺序放进一个代码块，段与段空行分隔（TTS不读空行），倒计时位、亮牌位留给剪辑，块内零引号零杂符。
2. **分题画面表**：每题一行，写清图标/牌面/选项/小剧场分镜，选项文字只出现在画面里，口播默认不读选项（题型特例见各题技能）。
3. **答案速查**：一行，给剪辑对答案用。第五题标注：开放题无标准答案，牌面为XXX。
4. **史实核查清单**：本期用到的每个牌面细节+怎么验证，评论区被杠时直接引用。
最后加一行热点·可换提示：开场钩子和题四处境可在发布当天换成更火的热搜，句式不动、牌义不动。

## 口播TTS铁律（每句都要过这一关）

口播是直接喂给TTS的纯文本，读错一个符号整条音频重录，这是第一死线。
1. 全程零引号：中英文单双引号、书名号、括号都不行。
2. 禁波浪号、省略号、破折号、冒号。标点只用逗号、句号、问号、顿号。
3. 数字全写汉字（五秒、七十八张、过了一百）。选项字母A B C D可以用。
4. 问句必须问号结尾，TTS自动读升调。
5. 生僻字换白话：螯虾写成小龙虾。
6. 段与段空一行。倒计时位、亮牌位留给剪辑，口播文本里不出现这些词。
7. 禁用词：卡在哪里。禁用抽象时间词开场（热的时候、冷的时候这类），情绪只用具体的事写。
8. 空场面话零容忍（别瞎蒙、我不拦着、慌了吧这类没有信息量的硬凶话）。每句话要么装一个真细节，要么埋一个钩子。
9. 语气：大白话短句，亲和带一点点撩，啦、呢、嘛少量点缀，不说教。受众约八成是二十五到三十五岁的都市女性打工人。
