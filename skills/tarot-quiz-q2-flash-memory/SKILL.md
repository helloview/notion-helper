# 题型二：三秒记忆

## 定版范例

画面：太阳牌完整展示三秒后整张消失，弹出问题和选项：A 白马 / B 狮子 / C 骆驼 / D 什么都没骑。
口播：第二题考你的记性力，这张太阳牌只给你看三秒。
（亮牌三秒，牌消失后）
口播：那么问题是，牌面中的小孩骑的是什么，你有五秒时间作答。
答案口播：答案是A白马，如果答错了没关系，证明你还得练。

注意：用户原稿的问题句带引号，已按TTS铁律剥除。本题型的问题句直接说，任何引号都不加。

## 出题公式

1. 口播先预告只看三秒，再亮牌（亮牌三秒是画面的事，口播不数秒）。
2. 问一个**看过但没注意**级别的细节。太明显（太阳牌上有太阳吗）没意思，太刁钻（第几根光芒）招人烦，问点要落在扫一眼记不住、看两眼能记住的层级。
3. 四选项进画面，口播不读选项。作答五秒。
4. 答案口播 = 答案是X + 软安抚（定版：如果答错了没关系，证明你还得练） + 可选一句反直觉补刀（例：对，死神骑的偏偏是一匹白马）。补刀是这个题型的验证冲动来源，能用就用。

## 细节四问法（每期换一类，别一直问骑什么）

数量、颜色、动作、位置。

## 记忆题牌库（问点 → 正确答案 → 好干扰项；用前逐条web核实）

1. 太阳（首发已用）：小孩骑的是什么 → 白马 → 狮子、骆驼、什么都没骑
2. 死神：骑的马是什么颜色 → 白色 → 黑色、红色、棕色。反直觉王牌，答案口播必补一刀
3. 愚人：脚边跟着什么 → 一只小白狗 → 黑猫、狼、什么都没有
4. 魔术师：桌上摆了几样东西 → 四样 → 三样、五样、六样
5. 星星：女子手里拿着几个水壶 → 两个 → 一个、三个、没拿东西
6. 高塔：有几个人往下掉 → 两个 → 一个、三个、没有人
7. 女祭司：两根柱子什么颜色 → 一黑一白 → 都是白的、都是黑的、一红一蓝
8. 正义：哪只手举着剑 → 右手 → 左手、双手、没有剑
9. 圣杯十：天上是什么 → 彩虹 → 太阳、乌云、月亮
10. 命运之轮：轮子顶上蹲着什么 → 狮身人面兽 → 猫、天使、皇冠
11. 隐者：灯里装的是什么 → 一颗星星 → 蜡烛、火苗、月亮。反直觉好点，灯里点的不是火
12. 节制：天使的脚踩在哪 → 一只脚水里一只脚岸上 → 都在水里、都在岸上、悬在空中

## 干扰项规则：合理但错

错误项要贴这张牌的气质才骗得到人。太阳牌配狮子（阳光猛兽联想）是好干扰项，配企鹅就是白给。每题至少一个强迷惑项加一个搞笑项（什么都没骑属于搞笑项，负责弹幕）。

## 变体（隔几期换一次玩法）

1. 双牌找不同：两张牌同时闪三秒，问哪张里有某元素。
2. 局部闪现：只闪牌面上半区，问下半区有什么，答案是根本没给你看，整活题，一期最多一次。
3. 颜色专场：闪一张彩色牌，问某物颜色。

## 查重

每期换牌换问点。首发期已用：太阳。

## 口播TTS铁律（每句都要过这一关）

1. 全程零引号：中英文单双引号、书名号、括号都不行。
2. 禁波浪号、省略号、破折号、冒号。标点只用逗号、句号、问号、顿号。
3. 数字全写汉字（三秒、五秒、四样）。选项字母A B C D可以用。
4. 问句必须问号结尾，TTS自动读升调。
5. 倒计时位、亮牌位留给剪辑，口播文本里不写。
6. 空场面话零容忍（别瞎蒙这类没有信息量的硬凶话），每句话要么装一个真细节，要么埋一个钩子。
7. 语气：大白话短句，亲和带一点点撩，啦、呢、嘛少量点缀。受众约八成是二十五到三十五岁的都市女性打工人。
