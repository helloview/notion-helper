# 题型四：读牌义（这张牌到底想说什么）

## 这道题为什么存在（先想清楚再出题）

前三题考的都是表层：看图（题一）、记性（题二）、认名字（题三）。第五题是完全开放的解读。**第四题卡在正中间，考的是唯一一件事：你到底懂不懂一张牌真正的意思。** 它是从「懂图」到「懂牌」的台阶。

旧版做法（先想一句金句当答案，再随便盖两张牌上去）已废弃，原因就是牌没在干活，撤掉牌那句金句照样成立，懂牌的人觉得凭什么、不懂的人觉得莫名其妙。新版铁律：**牌是题眼，把牌拿掉这道题就不成立**。

## 出题公式

1. 选一张**牌义明确、而且经常被误读**的牌（库见下）。误读越普遍，这道题越有价值，因为答完观众会恍然大悟。
2. 配一个能让这张牌义落地的**具体生活处境**（两三句，扎心、有画面）。处境是为牌义服务的，不是随便找个惨事。
3. 问一句：这张牌到底想告诉你什么？（不是问哪句文案好听，是问牌义）
4. 四个选项都是**对这张牌的解读**，作答五秒。
5. 答案口播 = 答案是X + 一句点出牌面上的关键细节 + 一句点破真正牌义。**必须落在牌面证据上**，这样既有说服力，又天然带一股去查就知道是真的的劲儿。

单张牌是默认。只有当「两张牌搭在一起产生的化学反应」本身就是题目时，才用两张，而且那时答案考的是这个组合的合义，不是各自一句话。

## 选项设计（四件套，每个都是一种读法）

- **正解**：这张牌真正的核心牌义，用一句能让人带走的话说出来。
- **字面吓人项**：把牌当成它字面上最可怕的样子（死神=真的要死人、高塔=纯灾难、恶魔=邪灵作祟）。这是最常见的误读，也是最诱人的错项。
- **甩锅外部项**：把牌义解释成都是别人/环境的错，而这张牌其实指向的是你自己（宝剑八最典型）。
- **反向项**：跟真牌义正好相反的读法。
正解位置每期换，别老放某个字母，答案口播报对应字母。

## 读牌义牌库（处境 → 正解牌义 → 常见误读=好干扰项；用前逐条web核实）

1. **宝剑八**（示范）：处境=想改变却迈不出第一步。正解=困住你的绳子是你自己系的，眼罩一摘就能走。误读=有人在背后困着你 / 时机没到只能等 / 大祸临头。牌面证据：蒙眼、松绑、脚边就是路。
2. **高塔**：处境=稳定的东西突然塌了（分手、被裁、计划泡汤）。正解=旧结构崩塌是为了逼出重建，留不住的本就该走。误读=纯粹的灾难和霉运。牌面证据：闪电劈塔、王冠飞落、两人坠落。
3. **死神**：处境=一段关系或一个阶段走到头。正解=结束与转化，旧的死去新的才生。误读=真的有人要死。牌面证据：骷髅骑白马、身后升起的太阳、跪着的人。
4. **恶魔**：处境=明知不好却离不开（烂关系、上瘾、执念）。正解=捆住你的是欲望和执念，而且链子是松的，你随时能挣脱。误读=有邪灵在害你。牌面证据：两人脖子上的链子松松垮垮，其实能取下来。
5. **月亮**：处境=看不清、心里发慌、疑神疑鬼。正解=眼下就是不明朗，别急着下判断，走过去再说。误读=一定有人在骗你 / 要出大事。牌面证据：小龙虾爬出水面又缩回、狗和狼朝月亮嚎、两塔之间一条幽路。
6. **星币四**：处境=攥着钱和安全感不敢松手，日子却越过越紧。正解=抓得太紧本身就是问题。误读=你就该更节俭。牌面证据：死死抱住钱币、头上脚下都压着钱。
7. **宝剑十**：处境=感觉被捅到底、彻底完了。正解=已经到最低点，接下来只会天亮，结束也是解脱。误读=后面还有更惨的。牌面证据：背上插满十把剑、远处天边已经泛白。
8. **命运之轮**：处境=正倒霉/正得意。正解=风水轮流转，别把此刻当永远。误读=命里注定改不了。牌面证据：巨轮、往下爬的蛇、四角看书的生物。
9. **倒吊人**：处境=卡住了、动不了、被迫停下。正解=换个角度看，这个停是主动的牺牲，倒过来才看得见新东西。误读=纯粹倒霉受困。牌面证据：单脚倒挂、表情平静、头顶一圈光。
10. **圣杯五**：处境=沉浸在失去里走不出来。正解=别只盯着打翻的三个杯子，回头还立着两个。误读=一切都没了。牌面证据：低头看倒地的三杯、身后还立着两杯。

## 处境域（给牌义找落点，每期换域）

职场（裸辞、offer二选一、被穿小鞋）、感情（分手、暧昧不明、复合与否）、家庭（和父母冷战、催婚）、自我（三十岁焦虑、拖延、自我怀疑）、金钱（借钱、冲动消费后悔）、人生岔路（去大城市还是回家、要不要赌一把）。选处境的唯一标准：能不能让这张牌的牌义精准落地。

## 查重

牌、处境每期换，不与往期重复。首发期第四题旧版用的是老板画饼（圣杯七+星币八，选字幕玩法，已废弃）；若上一期演示用过宝剑八读牌义，本期换别的牌。

## 口播TTS铁律（每句都要过这一关）

1. 全程零引号：中英文单双引号、书名号、括号都不行。
2. 禁波浪号、省略号、破折号、冒号。标点只用逗号、句号、问号、顿号。
3. 数字全写汉字。选项字母A B C D可以用。
4. 问句必须问号结尾，TTS自动读升调。
5. 生僻字换白话（螯虾写成小龙虾）。倒计时位留给剪辑，口播里不写。
6. 禁用词：卡在哪里。禁抽象时间词开场，处境只用具体的事写。
7. 空场面话零容忍，每句要么装一个真细节，要么埋一个钩子。
8. 答案口播必须落在牌面证据上，不许空讲道理，也不许在这里长篇解牌（点到即止，两句以内）。
9. 语气：大白话短句，亲和带一点点撩，不说教不毒鸡汤。受众约八成是二十五到三十五岁的都市女性打工人。
