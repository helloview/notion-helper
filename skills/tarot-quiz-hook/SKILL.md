# 开场钩子

## 定版

听说你会解读塔罗？来试试你的实力，总共五题，看看你到底有多牛。

## 钩子公式

挑衅（听说你会X） + 立赌局（总共N题） + 可炫耀的承诺（看看你有多牛）。

铁律：
1. **第一句就是钩子本身，零铺垫**。任何“大多数人觉得”式前摇，人还没听完就划走了。
2. 二十五字以内，约六秒。
3. 疑问句用问号收，TTS读出升调，挑衅感是问号给的。
4. **承诺词闭环**：开场的“牛”必须在第五题被回收（这题答得好，才算真的牛）。换了钩子里的承诺词，结尾闭环句同步改，不许只改一头。
5. 画面配大字挑战文案压屏即可，不抢口播。

## 四类扩展方向（每期换着用，别连用同一类）

**一、挑衅型**（定版所属）：质疑对方实力，激起不服。
- 都说自己懂塔罗？来，五道题，看看你几斤几两。
- 塔罗十级学者是吧？五道题，全对的评论区报到。

**二、淘汰型**：提前预告淘汰点，制造闯关感。
- 五道塔罗题，据说第三题就能筛掉一半人，你先别急着划走。
- 用这类时，被点名的那道题必须真是当期最阴的一题，说到做到。

**三、组队转发型**：把钩子对准观众身边的人，天然带转发。
- 你身边那个天天研究塔罗的朋友呢？把这条转给她，五题见真章。

**四、悬念型**：用一个真实到离谱的细节吊住人。
- 塔罗牌里藏着一只小龙虾，你信吗？五道题，第一题就是它。
- 用这类时，第一题必须真考这个细节，细节必须web核实为真。

**数据型慎用**：昨天这五道题，难倒了整个评论区。数据必须真实，首期和数据平平的时期禁用，编数据等于人设自爆。

## 热点·可换

发布当天若有更火且贴题的热搜（情感、职场、游戏类优先），可把第一句换成热点问句再接立赌局，内核句式不动。输出时标注 热点·可换。

## 口播TTS铁律（每句都要过这一关）

1. 全程零引号：中英文单双引号、书名号、括号都不行。
2. 禁波浪号、省略号、破折号、冒号。标点只用逗号、句号、问号、顿号。
3. 数字全写汉字（五题、五秒）。
4. 问句必须问号结尾，TTS自动读升调。
5. 空场面话零容忍，每句话要么装一个真细节，要么埋一个钩子。
6. 语气：大白话短句，亲和带一点点撩，不说教。受众约八成是二十五到三十五岁的都市女性打工人。
