# 题型三：找出假牌

## 定版范例

画面：四张牌背朝上排开，牌下标注：A 倒吊人 / B 女祭司 / C 预言家 / D 命运之轮。
口播：第三题，下面这四张牌的名字，有一个是我瞎编的，给你五秒看看你记得不。
答案口播：答案是C，一个明明可以预言的工具，却没有预言家。

## 出题公式

1. 三真一假。**假名必须比真名听起来更正常，真名专挑最怪的当诱饵**，观众的第一直觉会扑向真牌，这道题的乐趣就在这。
2. 四个名字进画面标注，口播默认不读名字（定版做法）。可选变体：想让纯听音频的人也能玩，就快速念一遍四个名字，念名字时匀速，别在假名上加重音。
3. 作答五秒。
4. 答案口播 = 答案是X + 一句反差包袱。造句公式：一个X的东西，偏偏没有Y。示例：一副能算命的牌，偏偏没有占卜师。一套里面有恶魔有死神的牌，偏偏没有诅咒。

## 编假名五套路（每期换套路）

1. 职业型：预言家（首发已用）、占卜师、驯兽师、守墓人、炼金术士
2. 蹭真牌型（最阴）：审判官（真牌是审判）、命运女神（真牌是命运之轮）、月神（真牌是月亮）、正义使者（真牌是正义）
3. 概念情绪型：希望、复仇、孤独、永恒、诅咒
4. 数字越界型：圣杯十一、权杖零、星币十五。小牌数字只到十，往上全是编的
5. 体系陷阱型（慎用）：圣杯公主。韦特体系宫廷牌叫侍从，公主是托特牌的叫法。用它时答案口播必须补一句说明体系差异，不然懂托特的观众会来杠

## 真牌诱饵榜（听起来最像编的真牌，按迷惑度排）

倒吊人、女祭司、节制、审判、教皇、隐者、力量、世界。
每期从榜上挑两到三张当诱饵，搭一张正常真牌（命运之轮、星星、太阳这类）稳住阵型。

## 变体（隔几期换一次）

1. 归属陷阱：四个名字全是真牌，混入一张小阿卡纳（比如圣杯皇后），问哪张不属于大阿卡纳。
2. 别名陷阱：愚人也常被叫小丑，女祭司也译女教皇，皇后也译女皇。可以出：下面哪个是这张牌的标准名。别名不算假牌，出题时口径要清楚。

## 大阿卡纳二十二张核对名单（出题前逐字核对，一个都不能记错）

愚人、魔术师、女祭司、皇后、皇帝、教皇、恋人、战车、力量、隐者、命运之轮、正义、倒吊人、死神、节制、恶魔、高塔、星星、月亮、太阳、审判、世界。

## 查重

假名、诱饵组合每期全换。首发期已用：预言家（诱饵：倒吊人、女祭司、命运之轮）。

## 口播TTS铁律（每句都要过这一关）

1. 全程零引号：中英文单双引号、书名号、括号都不行。
2. 禁波浪号、省略号、破折号、冒号。标点只用逗号、句号、问号、顿号。
3. 数字全写汉字（五秒、二十二张）。选项字母A B C D可以用。
4. 问句必须问号结尾，TTS自动读升调。
5. 倒计时位留给剪辑，口播文本里不写。
6. 空场面话零容忍，每句话要么装一个真细节，要么埋一个钩子。
7. 语气：大白话短句，亲和带一点点撩，啦、呢、嘛少量点缀。受众约八成是二十五到三十五岁的都市女性打工人。
