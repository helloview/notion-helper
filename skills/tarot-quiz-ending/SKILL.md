# 结尾CTA

## 定版

把你的解读发在评论区，一句话也行。下期我挑几条一条一条讲，想被翻牌的先关注。评论区见。

## 结尾公式（四件事，一件都不能少，也不许加第五件）

1. **降门槛**：一句话也行。评论门槛降到地板，长短都收。
2. **UGC飞轮**：下期我挑几条一条一条讲。观众的评论就是下一期的内容，这句是整个系列的发动机。
3. **关注挂理由**：想被翻牌的先关注。关注必须给一个具体的好处，禁止裸求关注、求赞求藏求转一串报菜名。
4. **利落收**：评论区见。三到五个字收死，不加任何一个字。

## 铁律

1. **绝不煽情**。一到结尾就内省升华，完播率直接掉，这是整个账号体系的血泪教训。结尾要干脆、向外、能引评论。
2. **只求一个核心动作**：评论。关注是搭在评论便车上的，不单独求。
3. 结尾紧贴题型五的问句，中间不许插任何总结陈词（今天这五题怎么样呀这类全删）。
4. 承诺必须兑现：说了下期挑评论讲，下期就真讲，翻牌机制是账号信用。

## 变体库（每隔几期换一条，别一条用到死）

1. 定版（见上）。
2. 牛字闭环版：前四题全对不算牛，第五题能说圆的才算。写在评论区，下期我挑几条一条一条讲，想被翻牌的先关注。评论区见。使用条件：开场必须用定版钩子（看看你到底有多牛），承诺词首尾呼应。
3. 上榜版：你的解读，下期可能被几万人看到。发评论区，一句话也行。先关注，别错过你自己上榜。评论区见。
4. 从众版：评论区已经有人开始写了，你也来一句。下期逐条讲，想被点名的先关注。下期见。使用条件：评论区确实有热度之后才能用，冷启动期禁用，说了没有就是自爆。
5. 嘴替版：别在心里答，发出来。下期我一条一条讲，想被翻牌的先关注。评论区见。

## 口播TTS铁律（每句都要过这一关）

1. 全程零引号：中英文单双引号、书名号、括号都不行。
2. 禁波浪号、省略号、破折号、冒号。标点只用逗号、句号、问号、顿号。
3. 数字全写汉字。
4. 问句必须问号结尾。
5. 空场面话零容忍，每句话要么装一个真细节，要么埋一个钩子。
6. 语气：大白话短句，干脆向外，不煽情不说教。受众约八成是二十五到三十五岁的都市女性打工人。
